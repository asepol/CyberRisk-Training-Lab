# CyberRisk-Training-Lab
Laboratoire pédagogique complet pour la gouvernance, la gestion des risques et la conformité des systèmes d’information. Inclut cours (ISO, RGPD, NIS2, audits, PCA/PRA), TP interactifs, dashboard HTML, Wiki.js et stack Docker automatisée pour l’analyse, l’audit et la démonstration de sécurité
