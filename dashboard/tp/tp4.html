TP4 – Audit Technique (Juice Shop)

Contexte

Juice Shop est une application volontairement vulnérable. Vous devez réaliser un audit technique et identifier une vulnérabilité exploitable.

Objectifs

  • Réaliser un scan technique
  • Identifier une vulnérabilité
  • Analyser l’impact métier

Ressources

  • Juice Shop
  • DevTools du navigateur (onglet Network)

Tâches à réaliser

  1. Analyser les requêtes HTTP dans l’onglet Network.
  2. Identifier un endpoint non protégé.
  3. Exploiter une vulnérabilité (ex : Broken Access Control).
  4. Capturer la requête vulnérable.
  5. Documenter l’impact métier.

Aide

  • Rechercher les endpoints commençant par /rest/.
  • Tester les accès sans authentification.

Livrables

Un mini-rapport d’audit (vulnérabilité, preuve, impact, recommandation).
