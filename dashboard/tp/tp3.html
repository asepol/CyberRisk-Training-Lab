TP3 – Méthodes EBIOS / OCTAVE / MEHARI

Contexte

Vous devez analyser un scénario d’attaque sur Nextcloud en appliquant trois méthodes d’analyse de risques : EBIOS RM, OCTAVE et MEHARI.

Objectifs

  • Comprendre les différences entre les méthodes
  • Appliquer EBIOS RM sur un scénario
  • Comparer OCTAVE et MEHARI

Ressources

  • Wiki juridique
  • Nextcloud

Tâches à réaliser

  1. Identifier les événements redoutés (ex : fuite de données Nextcloud).
  2. Définir les sources de menace (interne, externe, malveillante, accidentelle).
  3. Évaluer les scénarios d’attaque selon EBIOS RM.
  4. Appliquer OCTAVE : actifs → menaces → risques.
  5. Appliquer MEHARI : scénarios → impacts → mesures.

Aide

  • EBIOS RM : 5 étapes (contexte, événements redoutés, scénarios, mesures, risques).
  • OCTAVE : orienté organisation et actifs.
  • MEHARI : basé sur des catalogues de scénarios.
