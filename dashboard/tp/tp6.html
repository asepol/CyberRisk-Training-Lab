TP6 – PRA / PCA (Nextcloud + MinIO)

Contexte

Nextcloud tombe en panne. Vous devez restaurer les données à partir d’une sauvegarde MinIO.

Objectifs

  • Comprendre PRA/PCA
  • Réaliser une sauvegarde
  • Restaurer un service

Ressources

  • Nextcloud
  • MinIO Console

Tâches à réaliser

  1. Uploader un fichier dans Nextcloud.
  2. Sauvegarder /var/www/html/data dans MinIO.
  3. Supprimer le fichier dans Nextcloud.
  4. Restaurer depuis MinIO.
  5. Vérifier la récupération.

Aide

  • Créer un bucket “nextcloud-backup”.
  • Utiliser l’interface MinIO pour uploader/télécharger.
