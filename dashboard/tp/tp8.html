TP8 – Aspects Légaux (RGPD)

Contexte

Vous analysez un traitement de données personnelles dans le SI (ex : Nextcloud).

Objectifs

  • Comprendre RGPD
  • Identifier les risques juridiques

Ressources

  • Wiki juridique

Tâches à réaliser

  1. Choisir un traitement (ex : stockage de fichiers Nextcloud).
  2. Identifier les données personnelles.
  3. Déterminer la base légale (article 6 RGPD).
  4. Identifier les risques juridiques.

Aide

  • Article 6 RGPD : bases légales.
  • Article 32 : sécurité du traitement.
