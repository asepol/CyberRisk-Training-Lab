TP1 – Gouvernance & Réglementations

Contexte

Vous travaillez pour CyberCorp, une entreprise fictive devant se mettre en conformité avec les réglementations européennes (RGPD, NIS2) et les normes ISO 27001.

Objectifs

  • Identifier les obligations légales
  • Comprendre les exigences ISO 27001
  • Cartographier les risques juridiques

Ressources

  • Wiki juridique

Tâches à réaliser

  1. Identifier les données personnelles traitées par l’entreprise.
  2. Associer chaque traitement à une obligation RGPD.
  3. Identifier les articles NIS2 applicables.
  4. Lister les contrôles ISO 27001 pertinents.

Aide

  • Rechercher dans Wiki.js : “RGPD”, “registre”, “NIS2”.
  • Consulter les annexes ISO 27001 A.5 à A.18.
